THE STORY

I started breaking code before I knew how to write it. Coming from Indonesia, resources were just whatever I could scrape from documentation and open-source repos. That forged a habit: read the source, figure it out, build it better.

My approach sits somewhere between a traditional software engineer and a digital designer. I care about memory leaks as much as I care about bezier curves.

A fast site with terrible typography is a failure; a beautiful site that lags is a tragedy. I refuse to ship either.
